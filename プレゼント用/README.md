# IQ TETRIS QUEST プレゼントパッケージ

クイズ×テトリス×RPGの教育エンターテインメントゲームを作るためのパッケージです。

## 使い方

### Claudeでゲームを生成する

1. `要件定義書.md` の内容をClaudeにコピペ
2. 「この要件定義書に基づいてゲームを作成してください。通常版/カワイイ版を作ってください」と依頼
3. 生成されたコード（index.html, style.css, game.js）を保存
4. `assets/` フォルダの音声・画像ファイルを同じフォルダに配置
5. index.html をブラウザで開いてプレイ！

## フォルダ構成

```
プレゼント用/
├── README.md（このファイル）
├── 要件定義書.md（Claudeに渡す仕様書）
└── assets/
    ├── 共通/（両バージョンで使用）
    │   ├── iqtetrisquestロゴ.png
    │   ├── オープニング.mp3
    │   ├── BGM2.mp3
    │   ├── エンディング.mp3

    │   ├── クイズ正解1.mp3
    │   ├── クイズ不正解1.mp3
    │   ├── 攻撃2.mp3
```

## アセットの配置

### 通常版を作る場合
`assets/共通/` の中身をすべてゲームフォルダにコピー

### カワイイ版を作る場合
`assets/共通/` + `assets/カワイイ版専用/` の中身をすべてゲームフォルダにコピー

## カスタマイズのヒント

- **クイズを変更**: game.js 内の `quizData` オブジェクトを編集
- **敵キャラ変更**: game.js 内の `enemySettings` を編集
- **色を変更**: style.css のカラーコードを編集
- **BGM変更**: 同名のmp3ファイルを差し替え
