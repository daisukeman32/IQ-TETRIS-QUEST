# クイズテトリス

これは、クイズとテトリスを融合させた、新しいタイプの落ち物パズルゲームです。
ブロックが落ちてくる間にクイズに答えることで、ブロックを操作する権利を得られます。知識と判断力、そしてスピードが試されるゲームです。

## 主な機能

*   **レベル選択機能:**
    *   **対象者:** 「低学年」「高学年」「おとな」の3レベルから、出題されるクイズの難易度を選択できます。
    *   **スピード:** 「かんたん」「ふつう」「むずかしい」の3段階から、ブロックの落下スピードを選択できます。
*   **多彩なクイズ:** 対象者ごとに、質の高い様々なジャンルのクイズが出題されます。
*   **おたすけ機能:** 1ゲームに1回だけ、クイズに答えなくてもブロックを操作できる「おたすけボタン」が使えます。
*   **ゲームクリア:** スコアが100点に到達すると、特別な演出と共にゲームクリアとなります。
*   **サウンド対応:** BGMと効果音（SE）を再生する機能を搭載しています。（別途音声ファイルが必要です）

## 遊び方

1.  `index.html` ファイルをウェブブラウザで開きます。
2.  タイトル画面で「対象者」と「スピード」を選択します。
3.  「ゲームスタート！」ボタンを押して、ゲームを開始します。
4.  画面上部からブロックが落ちてくると同時に、画面右側にクイズが出題されます。
5.  ブロックが地面に着く前に、クイズに正解してください。
    *   **正解すると:** キーボードの矢印キー（← → ↓ ↑）でブロックを自由に操作できます。
    *   **不正解だと:** 「まちがい！」と表示され、正解するまでブロックの操作はできません。
6.  どうしても間に合わない時は、「おたすけボタン」が助けてくれます。
7.  スコア100点を目指して頑張りましょう！

## 必要な音声ファイル

このゲームのサウンド機能を完全に有効にするには、以下の音声ファイルを `index.html` と同じ階層に設置する必要があります。

*   `BGM.mp3`
*   `カーソル移動11.mp3`
*   `決定ボタンを押す.mp3`
*   `クイズ正解1.mp3`
*   `クイズ不正解1.mp3`
*   `おたすけ.mp3`
*   `大勢で拍手.mp3`
*   `「おめでとう」.mp3`
